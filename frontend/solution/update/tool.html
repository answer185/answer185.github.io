# 从零使用

步骤 命令 说明
1 transfer, 选择初始化 重新初始化新项目的src目录内容
2 transfer, 选择复制文件 从红圈项目中复制代码文件过去
3 transfer, 选择修正文件 修正新项目src目录中的文件，如routes改名为pages
4 package 生成依赖包的安装命令，并在新项目中安装(可选)
5 react 修改react, react-router的相关代码，使兼容新版本
6 path 修改模块路径的加载风格，统一为@风格的路径

# 命令

名称 作用
format eslint，prettier，stylelint的自动按目录格式化
react react升级
router 路由转umi配置，生成一个文件
transfer 复制文件到新项目中
path 相对路径转@
antd 检测位置并修改
info 显示系统的相关信息

# 选项

名称 作用
--source 指定处理的文件路径
--target 指定迁移的目标路径
